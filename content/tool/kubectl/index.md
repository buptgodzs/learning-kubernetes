## 用法

### 自动补全

执行下列命令，可以在bash下启动kubectl的自动命令补全，使用时特别的方便，强烈推荐：

```bash
source <(kubectl completion bash)
```

方便起见，可以直接修改`/etc/profile`，加入sourcek8s别名。

```bash
alias sourcek8s='source <(kubectl completion bash)'
```
